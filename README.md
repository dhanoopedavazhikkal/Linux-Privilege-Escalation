<h1>Linux Privilege Escalation</h1>


<h2>Description</h2>
Privilege escalation is a journey. There are no silver bullets, and much depends on the specific configuration of the target system. The kernel version, installed applications, supported programming languages, other users' passwords are a few key elements that will affect your road to the root shell.

This project was designed to cover the main privilege escalation vectors and give you a better understanding of the process. This new skill will be an essential part of your arsenal whether you are participating in CTFs, taking certification exams, or working as a penetration tester.This project was done with the help of tryhackme platform.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Bash</b>

<h2>Environments Used </h2>

- <b>Linux Ubuntu 14.04 LTS</b>
